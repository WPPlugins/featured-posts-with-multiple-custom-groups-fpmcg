This plugin allows you to create unlimited custom groups of featured posts.

== Description ==

This plugin allows you to create unlimited custom groups of featured posts. 
Admin can create unlimited groups and assign the posts under one or multiple groups. 
The widget section allows the admin to show these groups of featured posts. The widget section is hugely customizable to enter HTML and CSS to give the featured posts blocks the desired look. 
<strong>Please refer to the help section for more details.</strong>

== Installation ==

1. Unzip featured-posts-with-multiple-custom-groups-fpmcg.zip 
2. Upload folder `featured-posts-with-multiple-custom-groups-fpmcg` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to 'FPMCG' menu for further details and help section.

== Frequently Asked Questions ==

= How does this plugin defers from other Featured Posts plugins? =

Using this plugin you can create groups of featured posts and display them in various sections in your blog as widgets. So one single plugin will help you create multiple featured posts sections with different posts which can be completely configured by the admin.

= Can I place one post under several featured post groups? =

Yes. A single post can be defined under as many featured posts groups as the admin wants. 

= How many featured posts groups can be displayed in a single widget =

As many featured posts groups can be placed in a single widget. Hence a widget can display posts under a single featured posts group or multiple groups.

= Are there any configurable options for the widgets =

Yes. The widgets are highly customizable. From the title of the widget to adding HTML tags before and after every posts and each of its details, you can completely control the look of the widgets. Also you can add custom widget specific CSS.
